پروژه نهایی WebGIS

  دانشجویان: کیمیا غلامی  فاطمه حسینی زاده زهرا بیکعلی زاده

توضیح پروژه

در این پروژه یک سایت ساده WebGIS طراحی کردم که کاربر می‌تواند بعد از ثبت‌نام وارد نقشه شود و لایه‌های مکانی را مشاهده کند.

بخش‌های مختلف پروژه:

ثبت‌نام و ورود:
کاربر باید ابتدا ثبت‌نام کند و سپس وارد سایت شود.

نمایش نقشه:
برای نمایش نقشه از کتابخانه OpenLayers استفاده شده است.

اتصال به GeoServer:
لایه‌های مکانی در GeoServer منتشر شده‌اند و در سایت نمایش داده می‌شوند.

نمایش اطلاعات:
با کلیک روی هر عارضه در نقشه، اطلاعات آن (مثل نام کشور) نمایش داده می‌شود.

راهنمای اجرا:
1. ابتدا پایتون و کتابخانه Flask را نصب کنید. 
2.دستور python app.py را در ترمینال بزنید.
3. سپس در مرورگر آدرس http://127.0.0.1:5000 را باز کنید. 
رای اینکه لایه‌های GeoServer درست نمایش داده شوند، باید افزونه Allow CORS را در مرورگر Chrome فعال کنید.
